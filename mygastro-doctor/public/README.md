# MyGastro AI Website

A professional multi-page website for MyGastro AI — 
an AI-powered gastroenterology healthcare platform 
connecting clinics, doctors, and patients.

## Live Demo
https://srinu2504.github.io/mygastro-ai-website/

## Pages
- Home — Hero, stats, features, CTA
- About — Mission, story, team, values
- Features — All platform modules
- For Patients — Journey, plans, testimonials
- For Clinics — Onboarding, comparison table
- Contact — Form, FAQ, contact details

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Deployed on GitHub Pages
